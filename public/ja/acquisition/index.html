データの取得方法

撮影前の下準備 見出しへのリンク

チェックリスト:

  • 動画を撮影できることを確認してください。
  • 顕微鏡のステージに、回転角度を示すメモリがあることを確認してください。
  • 顕微鏡ステージが0°のときに、0°-180°を結んだ線分がカメラの撮影素子に平行であることを確認してください。

図：カメラの撮影素子とステージの整合性の確認。ステージ上に紙をおいて、モニター上で紙の像が水平になるようにカメラの角度を調整しましょう。

  • 顕微鏡の下方ニコルが東西方向の振動方向、上方ニコルが南北方向の振動方向であることを確認してください。

ワークフロー1: かんたんな組織解析の場合 見出しへのリンク

  1. 光学系をクロスニコルにしてください。
  2. ステージの回転角度を0°にしてください(下図a.)。
  3. 薄片をステージの0°-180°を結んだ線分と平行に配置してください(下図b.)。
  4. 動画撮影を開始してください。
  5. ステージを手でゆっくり360°回してください(下図c.)。回転方向を間違えないように。 回転は半時計周りです。回すスピードは、１回転を１分くらいかけてください。手で回すときに、スピードの変化や、途中で停止があっても問題ありません。
  6. 動画を停止してください。
  7. .aviか.mp4の形式で動画を保存してください。

図：回転動画の撮影方法。手でゆっくりとステージを回転させながら、0°から360°まで動画撮影をします。

図：回転動画。1分くらいかけてゆっくりとステージを回転させてください。

ワークフロー2: 石英のc軸方位解析 見出しへのリンク

このワークフローでは、石英の方位を、傾けた薄片と傾る前の薄片を比較することによって定量化します。このとき、 薄片おおよそ10°程度傾けたり、水平にしたりしつつ、薄片がずれないような工作をする必要がります。下記のような工作によって、薄片を傾ける機構を作成することができます。もちろん、傾ける角度が同じであればこの手法に従う必要はありませんし、ユニバーサルステージがあるのであれば、それを使用して10°の傾きをつけても問題ありません。また、厳密に10°である必要もありません。

今回は、ユニバーサルステージにアクセスできない実験室においても、なるべくずれの少ない傾けができる方法を提供します。

この概要は、下記のように、傾けたり水平にしたりする薄片の土台を作成することです。

用意するもの:

  • 薄片 50mm * 30mm
  • 薄片の厚さを0.03 mm以下にしてください。
  • 数10gのおもり
  • 平滑な3.5mm から4.5 mm程度の板を用意してください。縦横のサイズは数cmであればいいです。
  • 両面テープ

下準備:

  • 下図のように、板に1mm程度の細さに切った両面テープを貼り付けて剥離紙を剥がしてください。
  • 私は、新品の薄片を3枚重ねて、テープで２辺を止めた後、その上に両面テープをはって次のようなものをつくりました。
  • 観察視野をなるべく傾ける軸(＝両面テープ)の近くで、なおかつ板と重ならない部分に配置してください。

薄片を顕微鏡ステージ上で傾けている様子。六角レンチを重しとして代用しています。

図：傾いている状態

図：おもり(六角レンチ）を片側にのせて薄片が水平な状態

データの取得 見出しへのリンク

動画１ 見出しへのリンク

  1. 光学系をクロスニコルにして、鋭敏色検板を差し込んでください。
  2. ステージの回転角度を0°にしてください。
  3. 薄片をステージの0°-180°を結んだ線分と平行に配置してください。
  4. 薄片を傾けてください。
  5. モニター表示上で薄片の左端にピントがあっている状態にしてください。
Warning
extinction color 基本的に光学顕微鏡の写真や映像は反転した像が映し出されるので、実際は薄片の右端にピントがあっています。
  1. 動画撮影を開始してください。
  2. ゆっくりとステージと接眼レンズが離れる向きに ステージのピント10秒くらいかけて動かし、モニター表示上で薄片の右端にピントが合うまで動かして下さい(上図a)。
  3. 動画撮影を停止してください。

動画2 見出しへのリンク

  1. ステージの回転角度を45°にしてください。
  2. モニター表示上で薄片の左下端にピントがあっている状態にしてください。
Warning
extinction color 先程と同様に、基本的に光学顕微鏡の写真や映像は反転した像が映し出されるので、実際は薄片の右上にピントがあっています。
  1. ゆっくりとステージと接眼レンズが離れる向きに ステージのピント10秒くらいかけてを動かし、モニター表示上で薄片の右上にピントが合うまで動かして下さい(上図b)。
  2. 動画撮影を停止してください。

動画3 見出しへのリンク

  1. 優しく、傾いている薄片の上に重しをおいて、薄片を水平にして薄片にピントを合わせてください(上図c)。
  2. 動画撮影を開始し、すぐに停止してください(動画の長さはなんでも良い。1秒程度の動画で良い）。
Note
extinction color この動画は、傾いている動画と比較するための動画であり、一瞬だけ撮影できればOKです。

動画4 見出しへのリンク

  1. ステージの回転角度を0°に戻してください。
  2. 動画撮影を開始してください。
  3. ステージを手でゆっくり360°回してください(上図d.)。回転方向を間違えないように。 回転は半時計周りです。回すスピードは、１回転を１分くらいかけてください。手で回すときに、スピードの変化や、途中で停止があっても問題ありません。
  4. 動画を停止してください。

動画5 見出しへのリンク

  1. 鋭敏色検板を抜いてください。
  2. 動画撮影を開始してください。
  3. ステージを手でゆっくり360°回してください(上図e.)。先ほどと同様に、半時計回りに1分くらいかけて一回転ステージを回してください。
  4. 動画を停止してください。
  1. 最後に、すべての動画を保存してください。
